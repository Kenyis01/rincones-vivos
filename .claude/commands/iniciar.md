Estás iniciando un proyecto web nuevo basado en la plantilla Astro.

Ejecutá estos pasos en orden, sin preguntar confirmación entre ellos:

1. Instalá las dependencias con `npm install`
2. Verificá que la estructura de carpetas es correcta (que existen `src/pages/`, `src/layouts/`, `src/components/layout/`, `src/styles/`, `public/`, `docs/`, `.claude/rules/`)
3. Hacé un build de prueba con `npm run build` para confirmar que todo compila
4. Si el build fue exitoso, arrancá el servidor de desarrollo con `npm run dev`
5. Mostrá la URL local donde está corriendo el sitio

Al terminar, recordale al usuario que complete `docs/client-brief.md` y `docs/design-tokens.md` antes de empezar a construir.
